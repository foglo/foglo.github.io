foglo est hors ligne.

foglo était une application web conçue pour les nantais. Basé sur les données libres de data.nantes.fr, des services comme Yahoo! Weather, Twitter et Facebook, foglo se proposait d’être l’agrégateur de multiples données.

Il a été créé en 2012 par Martin Bonnier et Guillaume Clochard dans le cadre de l'appel à projets du concours "Rendez moi la ville plus facile" organisé par Nantes Métropole dans l'objectif de promouvoir les nouvelles données libres. foglo y a remporté le prix Etudiant.
